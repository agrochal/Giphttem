# Giphttem
CLI App which uses Giphy API to tweet random gifs.
## How to use?
* Make sure you have NodeJS installed, you can find it <a href="https://nodejs.org">here</a>.
* Run ```npm install``` in your command line to install the dependency packages.
* To start using bot get your **Twitter API** and **Giphy API** keys, then put them into ```configTMP.js``` file and rename it to ```config.js```.

* You are now ready to use **Giphttem** and tweet random gifs on your Twitter.
* Type ```node app.js``` to tweet random gif!
